---------------------------------------------------------------
VIBRAI: Control inteligente de dispositivos con voz y tecnología.
---------------------------------------------------------------

Descripción
-----------
VIBRAI es una aplicación revolucionaria que combina el poder de la inteligencia artificial con el control de dispositivos por voz. 
Este sistema integra las funcionalidades de asistentes virtuales como Alexa y Siri para ofrecer una experiencia fluida y unificada.

Características
---------------
✨ Compatibilidad total con comandos de voz para Alexa y Siri.  
✨ Control centralizado de dispositivos inteligentes.  
✨ Personalización avanzada de comandos y respuestas.  
✨ Interfaz intuitiva y de fácil uso.  

Requisitos
----------
🖥️ Python 3.x  
📂 Dependencias (ver `requirements.txt`)  
⚙️ Configuración activa de Alexa y Siri  

Instalación
-----------
1. **Clona este repositorio**:
   ```bash
   git clone https://github.com/navas98/VIBRAI.git
   ```
2. **Accede al directorio del proyecto**:
   ```bash
   cd VIBRAI
   ```
3. **Instala las dependencias**:
   ```bash
   pip install -r requirements.txt
   ```

Uso
---
1. Asegúrate de que tus dispositivos Alexa y Siri estén configurados y activos.  
2. Ejecuta la aplicación principal:
   ```bash
   python main.py
   ```
3. Sigue las instrucciones para sincronizar los asistentes y utilizar comandos personalizados.
