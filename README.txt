
Alexa_Siri
==========

Descripción
-----------
Alexa_Siri es una aplicación que permite integrar las funcionalidades de los asistentes virtuales Alexa y Siri, ofreciendo una experiencia de usuario unificada y mejorada para controlar dispositivos inteligentes y realizar tareas automatizadas.

Características
---------------
- Compatibilidad con comandos de voz para Alexa y Siri.
- Control centralizado de dispositivos inteligentes.
- Personalización de comandos y respuestas.

Requisitos
----------
- Python 3.x
- Dependencias adicionales listadas en `requirements.txt`.
- Configuración activa de Alexa y Siri en dispositivos compatibles.

Instalación
-----------
1. Clona este repositorio:
   ```
   git clone https://github.com/navas98/Alexa_Siri.git
   ```
2. Accede al directorio del proyecto:
   ```
   cd Alexa_Siri
   ```
3. Instala las dependencias necesarias:
   ```
   pip install -r requirements.txt
   ```

Uso
---
1. Asegúrate de que Alexa y Siri están configurados y activos.
2. Ejecuta el programa principal:
   ```
   python main.py
   ```
3. Sigue las instrucciones que aparecerán en pantalla para sincronizar los asistentes y utilizar los comandos.
